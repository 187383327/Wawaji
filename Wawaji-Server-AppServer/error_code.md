# 1. 通用错误码

|状态码|	含义|
|:------|:-----:|
|200|请求成功|
|400|请求参数错误|
|401|用户校验失败|
|402|参数非法|
|403|没有权限|
|407|访问太频繁，请稍后再试|
|500|服务器内部错误|
